Consulta en Discord

BattleMetrics usa un ID propio; con solo Steam64 la API a veces no acierta. El bot de Discord (Hexaytron) suele publicar el link correcto: el servidor también relee los últimos mensajes del canal configurado para encontrar ese link y tu Steam.

/player identifier: STEAMID
ABRIR DISCORD

Después de que Hexaytron responda en el canal, tocá Reintentar: el servidor busca en el historial del canal el link BM que coincida con tu Steam.

Vincular BattleMetrics Manual

Si no se encontró el ID automáticamente, abrí Atlas o BattleMetrics, copiá el link directo del perfil y pegalo acá.
