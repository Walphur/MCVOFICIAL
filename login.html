CONTROL ROOM

Usá ADMIN_PASSWORD para entrar. En Render también necesitás JWT_SECRET (mejor una clave larga distinta de la contraseña).

Si en la consola ves 405 o 404 en /api/auth/login, este dominio no está ejecutando Node (suele ser Cloudflare/GitHub Pages con el HTML solamente). Opciones: (A) en Render conectá el dominio personalizado al mismo Web Service que corre npm start; (B) una sola vez abrí login.html?api=https://TU-APP.onrender.com (queda guardado); (C) completá el meta name="mcv-api" en el <head> con esa URL de Render.
